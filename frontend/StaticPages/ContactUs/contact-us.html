Bug Reports

If you discover a bug or unexpected behavior, please include the following information whenever possible:

  • A brief description of the issue
  • Steps to reproduce the problem
  • Browser name and version
  • Device type
  • Screenshots or screen recordings (if applicable)

Providing detailed information helps us investigate and resolve issues more efficiently.

Security Issues

If you discover a potential security vulnerability, we encourage responsible disclosure.

Please avoid publicly sharing security-related issues until they have been investigated and, where appropriate, resolved.
